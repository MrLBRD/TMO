PRÉREQUIS
───────────────────────────────────────────────────────────────
• Windows 10 ou supérieur
• Webcam Logitech C270 (ou compatible)
• Google Chrome installé
• Visual C++ Redistributable (si erreur au lancement)
  → https://aka.ms/vs/17/release/vc_redist.x64.exe


INSTALLATION
───────────────────────────────────────────────────────────────

1. Lancez TMO_Setup.exe et suivez les étapes

2. Au premier lancement :
   - Cliquez sur CONFIG
   - Vérifiez la caméra détectée
   - Renseignez l'URL du site WooCommerce (https://...)
   - Enregistrez


CONFIGURATION DE L'EXTENSION CHROME
───────────────────────────────────────────────────────────────

Cette extension ferme automatiquement les anciens onglets TMO.

1. Ouvrez Chrome

2. Allez dans : chrome://extensions/

3. Activez "Mode développeur" (en haut à droite)

4. Cliquez "Charger l'extension non empaquetée"

5. Sélectionnez le dossier "chrome_extension" dans le dossier
   d'installation de TMO

6. L'extension est installée !

Raccourci clavier : Alt+Shift+L pour nettoyer manuellement

Note : après une mise à jour de TMO, rechargez l'extension dans
chrome://extensions/ (bouton de rechargement).


UTILISATION
───────────────────────────────────────────────────────────────

DÉMARRER UN ENREGISTREMENT :
• Scannez le QR code de la commande sous la caméra
  OU
• Entrez l'ID manuellement et cliquez START

L'overlay (petite fenêtre flottante) indique l'état :
• Gris "PRÊT" = En attente de scan
• Rouge "REC" + ID = Enregistrement en cours

ARRÊTER :
• Scannez le QR de la commande suivante (arrêt + nouveau départ auto)
  OU
• Cliquez STOP dans la fenêtre principale

DÉPLACER L'OVERLAY :
• Maintenez Ctrl + Clic et glissez


CALIBRATION DU SCANNER QR
───────────────────────────────────────────────────────────────

Si les QR codes sont mal détectés (éclairage difficile, caméra
ancienne), utilisez la calibration automatique :

1. Ouvrez CONFIG
2. Cadrez un QR code immobile dans l'aperçu (niveaux de gris)
3. Cliquez "Calibrer"
4. L'app teste automatiquement les meilleurs réglages
   brightness/contrast et met à jour les sliders
5. Cliquez Enregistrer

L'aperçu est affiché en niveaux de gris — c'est l'image exacte
que le scanner analyse, ce qui permet un ajustement précis.


STOCKAGE
───────────────────────────────────────────────────────────────

Les vidéos sont sauvegardées dans :
  %USERPROFILE%\TMO\output\

Format : YYYY/MM/DD/{ORDER_ID}.mp4

Rétention : 45 jours (configurable dans CONFIG)

Espace disque :
• Alerte orange : moins de 10 Go disponibles
• Alerte rouge   : moins de 2 Go disponibles


CONFIGURATION AVANCÉE
───────────────────────────────────────────────────────────────

Fichier de config :
  %APPDATA%\TMO\config.json

Pour réinitialiser : supprimez ce fichier

Logs de l'application :
  %APPDATA%\TMO\tmo.log


DÉPANNAGE
───────────────────────────────────────────────────────────────

PROBLÈME : Caméra non détectée
→ Branchez la webcam avant de lancer TMO
→ Testez avec l'application Windows Caméra
→ Dans CONFIG, cliquez "Rafraîchir"

PROBLÈME : QR code non scanné
→ Vérifiez l'éclairage (important pour C270)
→ Nettoyez la lentille de la webcam
→ Utilisez la calibration automatique (voir section ci-dessus)
→ Réduisez la zone de scan dans CONFIG (slider "Zone scan")

PROBLÈME : Chrome ne s'ouvre pas
→ Vérifiez dans CONFIG que Chrome est détecté
→ Installez Chrome si absent

PROBLÈME : Extension ne fonctionne pas
→ Vérifiez que le mode développeur est activé
→ Rechargez l'extension dans chrome://extensions/
